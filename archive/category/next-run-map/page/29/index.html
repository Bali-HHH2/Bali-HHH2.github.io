Last weeks run was a little bit far away for most people. But we still had a good turn out and after the run everyone agreed it was worth the drive. We enjoyed a Virgin Run at a Virgin Runsite set by not 1 but 2 old codgers. Even so it was a beautiful location, we had a good run and we ran a great circle.

This week Long Dong Silver will take us to Pura Ukur Ukuran in Pejeng, a location that we delightfully refer to as the Temple of Doom because of the 100 odd steps carved out of solid rock and worn shiny to get to the top from the river below. This is a extra-ordinary run site full of beautiful rice paddies, up and down river crossings and quaint villages. So come join us for another Run in Paradise with Bali Hash House Harriers 2
